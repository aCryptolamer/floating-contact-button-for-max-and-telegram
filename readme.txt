A lightweight floating contact button for WordPress with support for Telegram, WhatsApp, Facebook Messenger and MAX.

== Description ==

Floating Contact Button for Telegram, WhatsApp, Facebook Messenger and MAX adds a clean, modern floating contact button to your WordPress site.

The plugin allows visitors to quickly contact you via popular messengers using a compact floating menu that opens smoothly on click.

⚠️ Disclaimer:
This plugin is an independent project developed by the author and is not affiliated with, endorsed by, or officially connected to MAX Messenger, Telegram, WhatsApp, Facebook Messenger, or their respective owners.

== Key Features ==

* Floating contact menu with smooth open animation
* Supported messengers:
  * Telegram
  * WhatsApp
  * Facebook Messenger
  * MAX Messenger
* Modern glass-style design (Apple-like UI)
* One main button with expandable contacts
* Individual enable/disable for each messenger
* Drag & drop button reordering (order is saved automatically)
* Manual button positioning (Top / Right / Bottom in pixels)
* Lightweight and fast (no external libraries)
* Privacy-friendly (no tracking, no external servers)
* Easy configuration via WordPress admin panel
* Fully tested with the free AirTheme WordPress theme

== Installation ==

1. Upload the plugin files to the /wp-content/plugins/floating-contact-button-for-max-and-telegram directory, or install the plugin directly from the WordPress Plugins screen.
2. Activate the plugin through the Plugins menu in WordPress.
3. Go to Settings → Contact Button to configure your messenger links and button position.

== Frequently Asked Questions ==

= Is this plugin free? =
Yes. All current features are available for free.

= Is this an official plugin for Telegram, WhatsApp, Facebook Messenger or MAX? =
No. This is an independent plugin and is not affiliated with any messenger services.

= Does the plugin collect user data? =
No. The plugin does not collect, store or transmit any personal data.

= Can I disable individual messengers? =
Yes. Each messenger can be enabled or disabled independently in the settings.

= Can I change the position of the button? =
Yes. You can manually set top, right or bottom offsets in pixels. Leave fields empty to use the default position.

= Will more messengers be added? =
Possibly in future updates. The plugin is actively maintained.
