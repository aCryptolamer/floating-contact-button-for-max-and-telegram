# Floating Contact Button for MAX and Telegram

A lightweight floating contact button plugin for WordPress with support for Telegram, WhatsApp, Facebook Messenger and MAX.

## Features

- Floating contact button for WordPress
- Support for Telegram, WhatsApp, Facebook Messenger and MAX
- Simple settings page
- Custom icons
- Lightweight frontend code
- No external dependencies

## WordPress.org

Plugin page: https://wordpress.org/plugins/floating-contact-button-for-max-and-telegram/
